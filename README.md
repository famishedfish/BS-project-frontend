# Quick start

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

### Start project

```bash
npm start
```
  or if you want to use real data from backend
```bash
npm start : no-mock
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
